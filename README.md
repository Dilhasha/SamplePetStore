# SamplePetStore

A sample pet store application for demonstration purposes.

## Features

- Browse available pets
- Add pets to inventory
- Manage pet information

## Installation

### Prerequisites
- Ballerina Swan Lake 2201.x.x or later
- Java 11 or later

### Steps
1. Clone this repository
   ```bash
   git clone https://github.com/Dilhasha/SamplePetStore.git
   cd SamplePetStore
   ```
2. Install dependencies
   ```bash
   bal build
   ```
3. Run the application
   ```bash
   bal run
   ```

## Getting Started

Clone this repository and follow the setup instructions to get started with the Sample Pet Store application.
